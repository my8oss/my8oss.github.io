---

SQE Core is the core tool that converts .sqe files to html files.

Usage:
```bash
sqe-core [OPTIONS] --input <FILE>
```

Options:

| Option          | Description     |
|---------------------|------------------|
| `--input <FILE>`       | Input .sqe file to compile. |
| `--output <DIR>`          | Output directory (defaults to ./out) [default: out] |
| `-h`, `--help`         | Print help |
| `-V`, `--version`     | Print version |
