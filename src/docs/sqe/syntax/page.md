title: pages
description: generating/manages pages
---



Each Survey/Questionnaire can be devided into multiple pages, using ```@p``` flag:

```
@p <PAGE 1 NAME>
<PAGE 1>
@p <PAGE 2 NAME>
<PAGE 2>
```
For example:
```
@p page 1

@p page 2
```
