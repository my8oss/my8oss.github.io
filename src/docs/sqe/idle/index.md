SQE IDLE is SQE's simpler editor with live HTML preview.

![choice](/images/sqe/image-1.webp)

- To open a new file, click "New".

- Use the "Open" button to open SQE files.

- The "Save" button saves the file.

- To run the preview, click "Run".

- The "Live" preview shows the output in real time.

- Click "Info" to see information.

- Use the "Terminal" button to toggle the terminal output.
