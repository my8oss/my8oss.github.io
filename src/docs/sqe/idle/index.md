# SQE IDLE Editor Overview
SQE IDLE is a simple editor for `.sqe` files that provides a live HTML preview.

![SQE IDLE editor toolbar with New, Open, Save, Run, Live, Info, and Terminal buttons](/images/sqe/image-1.webp)

## Toolbar Buttons

- **New** – Create a new file.
- **Open** – Open an existing SQE file.
- **Save** – Save the current file.
- **Run** – Start the preview.
- **Live** – Toggle real-time preview updates.
- **Info** – Show file or editor information.
- **Terminal** – Show or hide the terminal output panel.
